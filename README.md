# XIII Files

A community unit archive, CP comparison tool, team builder, and APK-verified Carnival Banner Simulator for Tokyo Ghoul Awakening.

The simulator audit and exact rates are documented in `CARNIVAL-SIMULATOR-AUDIT.md`.

Presented by ICX (5081).

## GitHub Pages

This repository deploys automatically through `.github/workflows/pages.yml` whenever the `main` branch is updated.

The public package intentionally excludes APK files, extracted game configuration, development scripts, and private analysis artifacts.
